# Rat
### An alternative to tar made in rust

## Usage
```
Usage: rat [OPTIONS] --archive-file <ARCHIVE_FILE>

Options:
  -i, --input-files [<INPUT_FILES>...]  Files to archive only used for archiving not extracting
  -a, --archive-file <ARCHIVE_FILE>     Output file name for archiving and input file for extracting
  -x, --extract                         Extract from archive
  -c, --compress                        Compress archive
  -h, --help                            Print help information
  -V, --version                         Print version information
```
